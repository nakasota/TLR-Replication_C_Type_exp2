--- Comment #3 by DeedleFake ---
This already works exactly as you seem to be describing:

```go
package main

import (
	"fmt"
)

type Test[T any] []T

type Constraint interface {
	Test[int] | Test[string]
}

func DoSomething[T Constraint](v T) {
	fmt.Printf("%#v\n", v)
}

func main() {
	DoSomething(Test[int]{3, 2, 5})
	DoSomething(Test[string]{"a", "b", "c"})
	DoSomething(Test[float64]{3, 2, 5}) // Correctly fails to compile.
}
```

It doesn't seem likely to be particularly useful very often, though.
