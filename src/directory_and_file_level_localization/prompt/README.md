# C repository structure files

- This directory stores simplified repository structure files for the C project.
- Generate them with `../create_simplified_repo_structure.py`.
- Output directories:
  - `repository_structure_full/` (single file, full content)
  - `repository_structure_2000/` (split to ~2000 lines)
  - `repository_structure_1000/` (split to ~1000 lines)
  - `repository_structure_500/` (split to ~500 lines)
