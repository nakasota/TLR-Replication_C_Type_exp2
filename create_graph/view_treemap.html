About This Visualization

This treemap shows the results of the method_v6 evaluation on Go proposals. Each rectangle represents the complete directory path including src/ prefix (e.g., src/crypto/tls, src/runtime), with the size proportional to the total number of proposals in that directory. The rectangles are divided into red (accepted) and blue (declined) sections based on the method_v6 predictions.

Key Features:

  • Uses real data from method_v6 evaluation results
  • Shows complete directory paths including src/ prefix
  • Directory boundaries shown with black borders
  • Size represents total proposal count per directory
  • Color coding: Red = Accepted, Blue = Declined
  • 93 directories with varying acceptance rates
